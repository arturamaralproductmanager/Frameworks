This repository is organized with separate branches for Portuguese and English content.
How to navigate
The main branch may contain materials, names, or references mixed between both languages. This is intentional, since main works as the central organizational point of the repository.
To view the content separately by language:
Click the branch selector where main is displayed.
Open the list of available branches.
Choose the branch for your preferred language:
Português
English
Important
The branch-based language separation was designed to make navigation easier, improve readability, and provide more direct access to content in each language.
For a clearer analysis of the projects, always open the branch for the desired language before exploring the files.
Quick summary
main = central repository view
Portuguese branch = content in Portuguese
English branch = content in English
This structure was created to make the repository cleaner, more professional, and easier to understand.
