Este repositório possui conteúdos organizados em Português e Inglês em branches separadas.
Como navegar
A branch main pode apresentar materiais, nomes ou referências em ambos os idiomas. Isso acontece porque ela funciona como ponto central de organização do repositório.
Para visualizar o conteúdo de forma separada por idioma:
Clique no seletor de branch onde aparece main.
Abra a lista de branches disponíveis.
Selecione a branch correspondente ao idioma desejado:
Português
English
Importante
A separação por branch foi feita para facilitar a navegação, melhorar a leitura e permitir acesso mais direto ao conteúdo em cada idioma.
Se você deseja analisar os projetos com mais clareza, utilize sempre a branch específica do idioma antes de explorar os arquivos.
Resumo rápido
main = visão central do repositório
Branch em Português = conteúdo em português
Branch em Inglês = content in English
